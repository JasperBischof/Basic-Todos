# Basic-Todos

## Features

1. Adding todos to a list.
2. Checking todos in a list.
3. Deleting todos from a list.
4. Creating a todolist.
5. Deleting a Todolist.
6. Show todos based on active list.
7. Automatically select next list when active list gets deleted.
8. Store lists and todos in local storage.
9. Deletes all todos if the list they are contained gets deleted.
